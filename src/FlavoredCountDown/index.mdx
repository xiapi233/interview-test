## 使用 Vue 实现数字倒计时
### 📝 题目描述
请使用 Vue 框架实现一个风格化的倒计时效果，参考下面视频效果，倒计时从 20 开始每秒递减，直到 0 停止。

当倒计时数字小于 10 时，数字颜色应变为红色，以提醒用户。

<DemoVideo style="width: 50%" /> 

### 要求
1. 数字效果必须和视频一致，数字从 20 开始，每秒递减，直到 0 停止；
1. 实现方式不限，可以使用：
  1. 纯 CSS 绘制；
  2. SVG；
  3. Canvas；
2. 保持代码工整，逻辑清晰易读
