# 实现 CountdownButton 倒计时按钮组件

## 📝 题目描述

请实现一个 Vue 3 组件 `<CountdownButton />`，点击后进入倒计时状态，常用于“发送验证码”等场景。

## 🎯 功能要求

1. 点击按钮后开始倒计时（单位：秒），按钮变为禁用状态。
2. 倒计时期间，按钮文本变为「重新获取（N s）」。
3. 倒计时结束后，按钮恢复为原始插槽内容，允许再次点击。
4. 点击按钮时触发 `@click` 事件。

---

## 🧩 示例使用方式

```vue
<template>
	<CountdownButton :duration="5" @click="handleSendCode"> 发送验证码 </CountdownButton>
</template>

<script setup lang="ts">
	function handleSendCode() {
		console.log('验证码发送中...')
	}
</script>
```
