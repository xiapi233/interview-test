# 实现发布订阅系统（PubSub）

## 📝 请实现一个 PubSub 类，它支持以下方法：

-   `subscribe(event, callback)`: 订阅指定事件。
-   `publish(event, ...args)`: 发布事件，调用所有监听该事件的回调。
-   `unsubscribe(event, callback)`: 取消订阅指定事件的某个回调。
-   `clear(event?)`: 清除某个事件的所有回调，若不传参数，则清除所有事件。

## 🎯 要求：

-   类型安全（event 为 string 类型时，每种事件对应不同参数类型）
-   完善的类型定义
-   类型提示友好（能自动提示事件名字, 并且 TS 能推断出每个事件的回调参数）
-   支持多次订阅、取消订阅、清除等
-   运行 `pnpm test` 测试无错误

## 🧩 示例用法：

```ts
type Events = {
    click: [x: number, y: number]
    close: []
    data: [value: string]
}

const pubsub = new PubSub<Events>()

function onClick(x: number, y: number) { ... }
pubsub.subscribe('click', onClick)
pubsub.publish('click', 10, 20)
pubsub.unsubscribe('click', onClick)
```
