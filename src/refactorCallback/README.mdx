# 用 Promise 重构回调地狱

## 🎯 功能要求
用 Promise 重构回调地狱的代码，使其更易读和维护。

## 🧩 重构代码
```ts
function nestedTimeouts() {
	let t1 = setTimeout(() => {
		console.log(111)
		let t2 = setTimeout(() => {
			console.log(222)
			let t3 = setTimeout(() => {
				console.log(333)
				let t4 = setTimeout(() => {
					console.log(444)
					let t5 = setTimeout(() => {
						console.log(555)
					}, 500)
				}, 400)
			}, 300)
		}, 200)
	}, 100)
}
```

>
> 请在 `src/refactorCallback/index.ts` 完成代码编写.
